# Plan: 신규 기능 제안 (임팩트 분석)

## 개요
현재 앱(47+ 페이지, KR/US 듀얼 마켓)의 상태를 분석하여 가장 임팩트 있는 다음 기능을 제안한다.

## 현재 앱 분석

### 강점
- 포괄적인 KR/US 듀얼 마켓 지원
- AI 기반 분석 (스코어링, 보고서, 챗봇)
- 소셜 트레이딩 (아이디어, 리더보드)
- 배당 연구소 (스크리너, 시뮬레이터, 캘린더)
- 백테스트 엔진

### 약점/기회
- 모바일 UX 최적화 부족
- 알림 시스템 기초적 (인앱만, 푸시 없음)
- 데이터 시각화 단순 (기본 차트만)
- 사용자 간 상호작용 제한적

## 임팩트 순위별 제안

### 1순위: 실시간 알림 고도화 (High Impact)
- **현재**: 인앱 알림만, SSE 기반 단순 구현
- **제안**:
  - 가격 돌파 알림 (지지/저항선 기반)
  - 실적 서프라이즈 알림
  - 외국인/기관 대량 매수 알림
  - 알림 히스토리 & 통계 대시보드
- **임팩트**: 일일 리텐션 극대화, 재방문율 향상
- **난이도**: Medium

### 2순위: 포트폴리오 성과 분석 강화 (High Impact)
- **현재**: 기본 보유 종목 관리, 거래 기록
- **제안**:
  - 기간별 수익률 차트 (1W, 1M, 3M, YTD)
  - 벤치마크 대비 성과 (KOSPI, S&P500)
  - 섹터 분산도 분석
  - 세금/수수료 계산기
  - 월별 배당 수입 추이
- **임팩트**: 핵심 사용 가치 제공
- **난이도**: Medium

### 3순위: 인터랙티브 차트 강화 (Medium Impact)
- **현재**: lightweight-charts 기본 캔들차트
- **제안**:
  - 기술적 지표 오버레이 (RSI, MACD, BB 토글)
  - 드로잉 도구 (추세선, 피보나치)
  - 다종목 비교 차트
  - 호가/체결 실시간 표시
- **임팩트**: 트레이더 경험 개선
- **난이도**: High

### 4순위: PWA + 푸시 알림 (Medium Impact)
- **현재**: manifest.webmanifest 있으나 기본적
- **제안**:
  - 서비스 워커 기반 오프라인 지원
  - Web Push API 연동
  - 홈 화면 추가 안내
  - 앱 업데이트 알림
- **임팩트**: 모바일 경험 네이티브 앱 수준으로
- **난이도**: Medium

## 추천 우선순위
1. **실시간 알림 고도화** — 투자 앱의 핵심, 사용자 리텐션 직결
2. **포트폴리오 성과 분석** — "내 돈"에 대한 인사이트, 핵심 가치
3. **인터랙티브 차트** — 트레이더 UX, 경쟁력 확보
4. **PWA + 푸시** — 모바일 접근성, 알림 확장 기반
