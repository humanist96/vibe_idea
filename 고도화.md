## 1순위: 자연어 스크리너 (Natural Language Screener)

**현재**: 필터 UI로 수동 조건 설정
**개선**: 자연어로 종목 검색

- "PER 10 이하, 배당 3% 이상인 2차전지 관련주 찾아줘"
- "최근 한 달 외국인 순매수 상위 코스닥 소형주"
- "실적 서프라이즈 나온 저PBR 종목"
- GPT가 자연어 → 스크리너 필터 JSON 변환 → 결과 반환
- **활용 데이터**: 기존 screener API + chat intent 확장

---

## 2순위: AI 포트폴리오 닥터 (Portfolio Health Check)

**현재**: 마이페이지에서 보유 종목/수익률만 표시
**개선**: AI가 포트폴리오를 종합 진단하고 리밸런싱 제안

- 섹터 편중도, 종목 간 상관관계 분석
- 리스크 집중도 경고 (예: "반도체 비중 60% — 과도한 섹터 편중")
- AI 기반 대안 종목 제시 (유사 펀더멘털, 낮은 상관계수)
- 포트폴리오 점수 (분산성, 수익성, 안정성 각각 점수화)
- **활용 데이터**: portfolio API + AI scoring + sectors + 상관계수 계산

---

## 3순위: 실시간 뉴스 임팩트 알림 (News Impact Alert)

**현재**: 뉴스 단순 나열, 알림은 기본적
**개선**: AI가 뉴스를 실시간 분석하여 내 관심종목 영향도 평가

- 속보 발생 시 관심종목/포트폴리오 영향 자동 분석
- 임팩트 등급 (긍정/부정/중립 + 강도 1~5)
- "삼성전자 NAND 가격 인상 뉴스 → 관심종목 SK하이닉스에 긍정적 (임팩트 4/5)"
- 푸시 알림으로 즉시 전달
- **활용 데이터**: Google News + Finnhub news + watchlist + portfolio

---

## 4순위: 스마트 이상 감지 알림 (Anomaly Detection)

**현재**: 내부자 거래 폴링만 존재
**개선**: AI가 다차원 이상 신호를 감지하여 지능형 알림

- 비정상 거래량 급등 (평소 대비 3배 이상)
- 내부자 거래 패턴 변화 (집중 매수/매도)
- 대량보유 변동 신고 감지
- 기술적 지표 이상 (RSI 극단, 볼린저 밴드 이탈)
- 알림 피로도 관리: AI가 중요도 판단 후 정말 중요한 것만 알림
- **활용 데이터**: insider API + block-holdings + technical scoring + volume

---

## 5순위: AI 실적 프리뷰 & 리뷰 (Earnings Intelligence)

**현재**: 실적 서프라이즈 데이터만 사후 표시
**개선**: 실적 발표 전/후 AI 브리핑

- **프리뷰** (발표 전): 컨센서스 vs 최근 트렌드, 핵심 관전포인트, 서프라이즈 확률
- **리뷰** (발표 후): Beat/Miss 분석, 주가 영향 예측, 향후 전망
- 실적 캘린더와 연동하여 관심종목 실적 D-day 알림
- **활용 데이터**: earnings API + consensus + AI scoring + 뉴스 센티멘트

---

## 6순위: AI 종목 DNA 매칭 (Similar Stock Discovery)

**현재**: 종목 비교는 수동으로 2개 선택
**개선**: 한 종목의 "DNA"를 분석하여 유사 종목 자동 추천

- 펀더멘털 프로필 (PER, PBR, ROE, 성장률)
- 기술적 패턴 유사도
- 섹터/테마 겹침도
- "삼성전자와 DNA가 비슷한 종목 5개" → SK하이닉스, TSMC, 마이크론...
- 한국↔미국 크로스마켓 유사 종목 발견 (글로벌 피어 그룹)
- **활용 데이터**: screener 데이터 + AI scoring dimensions + sector/theme

---

## 7순위: AI 투자 일기 (Auto Investment Journal)

**현재**: 없음
**개선**: 하루 끝에 AI가 자동으로 투자 일기 생성

- 오늘 시장 요약 + 내 포트폴리오 성과
- 관심종목 주요 변동 사유 분석
- "오늘 삼성전자 -2.3%: 외국인 대량 매도 + 반도체 섹터 전반 약세"
- 주간/월간 리뷰 자동 생성 (성과 기여도 분석)
- 과거 판단 복기 기능 ("3개월 전 매수 시점 분석")
- **활용 데이터**: portfolio + daily report + market summary + investor flow

---

## 8순위: AI 리스크 레이더 (Personal Risk Dashboard)

**현재**: Fear & Greed Index만 존재
**개선**: 내 포트폴리오 관점의 개인화된 리스크 대시보드

- 매크로 리스크 (금리, 환율, 유가 방향성)
- 지정학 리스크 (AI 뉴스 분석 기반)
- 섹터 로테이션 신호 (내 보유 섹터의 자금 유출입)
- 밸류에이션 리스크 (과열/저평가 구간)
- 종합 리스크 점수 (1~100) + 레이더 차트 시각화
- **활용 데이터**: macro + fear-greed + sectors + valuation + portfolio

---

## 9순위: AI 시나리오 시뮬레이터 (What-If Analysis)

**현재**: 없음
**개선**: 매크로 변수 변경 시 포트폴리오 영향 시뮬레이션

- "금리 0.5% 인상되면 내 포트폴리오는?"
- "환율 1,400원 돌파하면 어떤 종목이 수혜?"
- "유가 20% 하락 시나리오"
- AI가 과거 데이터 기반 민감도 분석 + 시나리오별 예상 수익률
- 인터랙티브 슬라이더 UI로 변수 조절
- **활용 데이터**: macro API + FRED + portfolio + 과거 상관관계

---

## 10순위: AI 투자 코치 (Contextual Learning)

**현재**: 채팅에서 교육 intent는 있지만 기본적
**개선**: 사용자 수준과 포트폴리오에 맞춘 맞춤형 투자 교육

- 사용자 행동 기반 레벨 판별 (초보/중급/고급)
- 상황 맞춤 학습: "RSI가 뭔가요?" → 현재 보유 종목의 실제 RSI로 설명
- 실수 코칭: "고점 매수 패턴이 3회 감지됨 — 분할 매수 전략 제안"
- 주간 퀴즈/미니 레슨 (이번 주 시장 상황 기반)
- 학습 진도 트래킹 + 배지 시스템
- **활용 데이터**: chat history + portfolio + user behavior + market context
