### 1-2. 각 API 상세 분석

#### Finnhub (추천 1순위)
- **URL**: https://finnhub.io
- **무료 한도**: 60 calls/min, 일일 한도 없음
- **인증**: API Key (무료 발급)
- **데이터 범위**: 미국 주식, 일부 해외, 포렉스, 크립토
- **주요 엔드포인트**:
  ```
  GET /api/v1/quote?symbol=AAPL                    # 실시간 시세
  GET /api/v1/stock/profile2?symbol=AAPL            # 기업 프로필
  GET /api/v1/stock/candle?symbol=AAPL&resolution=D # OHLCV 차트
  GET /api/v1/company-news?symbol=AAPL              # 기업 뉴스
  GET /api/v1/search?q=apple                        # 종목 검색
  GET /api/v1/stock/metric?symbol=AAPL&metric=all   # 재무 지표
  GET /api/v1/calendar/earnings                      # 실적 캘린더
  WSS ws.finnhub.io                                  # 실시간 WebSocket
  ```
- **장점**: 가장 넉넉한 무료 한도, WebSocket 지원, 뉴스/검색/실적 캘린더 제공
- **단점**: 무료 히스토리 1년 제한, 상세 재무제표(분기별)는 유료
- **상용**: ToS 직접 확인 필요 (일부 디렉토리에 "Free, Commercial" 표기)

#### Twelve Data (추천 2순위)
- **URL**: https://twelvedata.com
- **무료 한도**: 800 calls/day, 8 calls/min
- **인증**: API Key (무료 발급)
- **데이터 범위**: 미국 무료, 90+ 해외 거래소는 유료
- **주요 엔드포인트**:
  ```
  GET /time_series?symbol=AAPL&interval=1day        # OHLCV 히스토리
  GET /quote?symbol=AAPL                             # 시세
  GET /price?symbol=AAPL                             # 현재가만
  GET /symbol_search?symbol=apple                    # 검색
  GET /earnings?symbol=AAPL                          # 실적
  GET /dividends?symbol=AAPL                         # 배당
  GET /statistics?symbol=AAPL                        # 통계
  WSS ws.twelvedata.com                              # WebSocket (8종목)
  ```
- **장점**: **상용 허용 명시**, 깔끔한 API, 기술지표 100+개, 검색 우수
- **단점**: 800/일은 트래픽 많으면 부족, 뉴스 미제공, 해외 거래소 유료
- **상용**: 공식 허용 (데이터 재배포/경쟁 서비스 구축은 금지)

#### Financial Modeling Prep (유료 권장)
- **URL**: https://financialmodelingprep.com
- **무료 한도**: 250 calls/day (개인 비상용만)
- **유료**: Starter $14/월 (상용 허용, 10,000 calls/day)
- **인증**: API Key
- **데이터 범위**: 150+ 엔드포인트, 가장 포괄적 재무 데이터
- **주요 엔드포인트**:
  ```
  GET /v3/quote/AAPL                                # 시세
  GET /v3/profile/AAPL                               # 기업 프로필
  GET /v3/historical-price-full/AAPL                 # 일봉 히스토리
  GET /v3/income-statement/AAPL                      # 손익계산서
  GET /v3/balance-sheet-statement/AAPL               # 대차대조표
  GET /v3/cash-flow-statement/AAPL                   # 현금흐름표
  GET /v3/stock_news?tickers=AAPL                    # 뉴스
  GET /v4/company-outlook?symbol=AAPL                # 올인원 개요
  GET /v3/search?query=apple                         # 검색
  ```
- **장점**: 재무제표 품질 최고, 올인원 엔드포인트, 실적/배당 상세
- **단점**: 무료=개인 비상용, 상용은 $14/월 필수
- **상용**: 유료 플랜부터 허용

#### 한국투자증권 OpenAPI
- **URL**: https://apiportal.koreainvestment.com
- **한도**: 초당 ~20건, 무료
- **조건**: 한투 증권계좌 필수 (한국 거주자)
- **해외 주식 지원**: 미국, 일본, 홍콩, 중국 등
- **엔드포인트**:
  ```
  /uapi/overseas-price/v1/quotations/price           # 해외 시세
  /uapi/overseas-price/v1/quotations/dailyprice       # 해외 일봉
  /uapi/overseas-stock/v1/trading/order               # 해외 주문
  ```
- **장점**: 실시간 데이터, 무료, 매매 연동 가능
- **단점**: 계좌 필수, 개인 앱키 기반 (공개 서비스로 불가), 3개월 미거래 시 해지

### 1-3. 추천 API 조합 전략

```
┌─────────────────────────────────────────────────┐
│                 InvestHub 해외주식                │
│                                                  │
│  [Finnhub 무료]          [Twelve Data 무료]      │
│  ├ 실시간 시세            ├ OHLCV 히스토리        │
│  ├ 기업 프로필            ├ 기술지표 (RSI, MACD)  │
│  ├ 뉴스                  ├ 종목 검색             │
│  ├ 실적 캘린더            ├ 실적/배당             │
│  └ WebSocket             └ 통계 데이터            │
│                                                  │
│  [FMP $14/월] (선택적 확장)                       │
│  ├ 상세 재무제표 (IS/BS/CF)                      │
│  ├ 올인원 기업 개요                              │
│  └ 과거 시가총액/배당 히스토리                    │
└─────────────────────────────────────────────────┘
```

**Phase 1 (무료)**: Finnhub + Twelve Data 조합
- Finnhub: 시세, 프로필, 뉴스, WebSocket
- Twelve Data: 히스토리, 기술지표, 검색, 배당

**Phase 2 (유료 확장)**: + FMP Starter ($14/월)
- 상세 재무제표, 올인원 기업 분석

**API 호출 예산 (Phase 1 기준)**:
- Finnhub: 60/분 = 3,600/시 = 일 약 86,400건 (넉넉)
- Twelve Data: 800/일, 8/분 (시세 캐시 5분 + 히스토리 캐시 1시간이면 충분)

## 5. 기술 제약 및 해결 전략

### 5-1. API 호출량 관리

```
문제: Twelve Data 800/일, Finnhub 60/분 제한
해결:
  1. 적극적 캐싱 (memory-cache.ts 패턴 재활용)
     - 시세: 장중 5분, 장외 1시간
     - 프로필: 24시간
     - 히스토리: 1시간
     - 뉴스: 15분
  2. 미국 장 시간 인식 (09:30~16:00 ET)
     - 장외 시간에는 캐시 TTL 연장, API 호출 최소화
  3. Finnhub을 실시간 시세 primary로, Twelve Data를 히스토리/검색용으로 분리
  4. 대시보드 상위 종목은 벌크 API 활용 (한 번에 여러 종목)
```

### 5-2. 한국어 지원

```
문제: 미국 기업 정보/뉴스는 영문
해결:
  1. 기업 설명: AI 번역 캐시 (OpenAI로 1회 번역 후 24시간 캐시)
  2. 뉴스 헤드라인: 원문 표시 (번역하면 의미 왜곡 우려)
  3. 섹터/산업명: 매핑 테이블 (Technology → 기술, Healthcare → 헬스케어)
  4. 재무 항목: Revenue → 매출, Net Income → 순이익 등 고정 매핑
  5. UI 라벨: 한국어 유지 (시가총액, 거래량, 등락률 등)
```

### 5-3. 통화 표시

```
미국 주식: $195.50 (USD 표기)
```

## 8. 핵심 판단 포인트

### Q1: 무료 API만으로 상용 서비스가 가능한가?

**가능하지만 제한적이다.**
- Finnhub + Twelve Data 조합으로 기본 시세, 차트, 뉴스, 검색, 실적 캘린더 제공 가능
- 상세 재무제표(손익계산서, 대차대조표)는 FMP 유료($14/월)가 사실상 필수
- 일 트래픽 1,000명 이하 서비스에서는 무료 API만으로 운영 가능

### Q2: 국내/해외를 하나의 앱에 통합할 것인가, 분리할 것인가?

**통합하되 모드 전환 방식을 추천한다.**
- 장점: 하나의 관심종목에서 KR+US 혼합 관리, AI 챗에서 자연어로 양쪽 질문
- 단점: 라우팅/데이터 계층 복잡도 증가
- 구현: URL 기준 `/us-stocks/...` 와 `/stock/...`로 분리하되, 사이드바에서 모드 전환

### Q3: 첫 출시에 어디까지 포함해야 하는가?

**Phase 1~2만으로도 MVP 출시 가능.**
- 시세 + 차트 + 검색 + 기업 개요 + 뉴스 = 기본적인 해외 주식 탐색 서비스
- 실적 캘린더와 AI 챗은 차별화 요소로 Phase 3~4에서 추가
- 재무제표는 유료 API 결정 후 Phase 5에서 확장
